# VG-Mapping: Variation-Aware 3D Gaussians for Online Semi-static Scene Mapping
Yicheng He, Jingwen Yu, Guangcheng Chen, Hong Zhang<br>

This repository contains the dataset and code for the paper VG-Mapping: Variation-Aware 3D Gaussians for Online Semi-static Scene Mapping, a online RGB-D 3D Gaussians mapping system tailored for semi-static scenes.
